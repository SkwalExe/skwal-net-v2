# Local Setup for Development

### Install the Requirements

```bash
pip install -r requirements.txt
```

### Create a .env file

```bash
cp .env.sample .env
# and edit it if necessary
vim .env
```

### Run the server

```bash
python3 skwal_net/manage.py runserver
```
